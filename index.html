Welcome to Anime Manager!

Anime Manager is an application built by Toby Horn for a college assignment at Full Sail University.

This application utilized an API created for a website called MyAnimeList.

Anime Manager is a way to create and track your own personal list of anime.

Note: Local Storage is being used to store user data, so clearning website data will delete your personal list.
